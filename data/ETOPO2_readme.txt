ETOPO2.nc contains the whole global elevation (and bathymetry) data in reasonable resolution. For higher resolution (ETOPO1.nc) see _etopo1_readme.txt


If there is no file ETOPO2.nc in this folder, download www.ngdc.noaa.gov/mgg/global/relief/ETOPO2/ETOPO2v2-2006/ETOPO2v2c/netCDF/ETOPO2v2c_f4_netCDF.zip (last modified 15-May-2007 15:23, size 85M) and rename it (after unzipping, if necessary) to ETOPO2.nc
